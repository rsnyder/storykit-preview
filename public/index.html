StoryKit Preview

Render a StoryKit markdown post instantly — no GitHub Pages build required.

Browser Bookmarklet Setup

1
Make sure your bookmarks bar is visible.
⌘⇧B on Mac  ·  Ctrl+Shift+B on Windows / Linux
2
Drag the button below to your bookmarks bar:
Preview on GitHub

drag to your bookmarks bar

3
Navigate to any .md file in your repository on GitHub, then click the bookmarklet to open a live preview in a new tab.

The bookmarklet only works on GitHub blob URLs for .md files — e.g. github.com/owner/repo/blob/main/path/to/post.md. Clicking it anywhere else shows an alert.

Direct URL

You can also open a preview by constructing the URL directly:

/owner/repo/path/to/file.md

Add ?branch=branch-name to preview a non-default branch (default: main).

Example: /rsnyder/storykit-starter/_posts/2026-01-10-monument-valley.md
